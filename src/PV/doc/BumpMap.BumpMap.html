Bump Mapper (BumpMap)

This filter implements a bump mapping effect on the geometry by modifying the point normals.
The BumpMap filter modifies the point normals so as to bump map the geometry, not only as a rendering effect.
Property Description Default(s) Restrictions
Input This property specifies the input to the BumpMap filter. This is the dataset to which the bump map effect will be applied.

Accepts input of following types:

  • vtkDataSet

The dataset much contain a field array (point) with 1 component(s).

The dataset much contain a field array (point) with 3 component(s).

SetScaleFactor The value of this property will be used as a multiplier for scaling the perturbation of the normals. 1.0
